1. Clone the repo:
       git clone https://github.com/arshabbir/movieapi
2. Go to movieapi directory
        cd movieapi
3. To test using docker please follow the below 
   a. Build the application docker image using the below command :
		sudo docker build . -t testapi
   b. Run docker compose file using below command
 	        sudo docker compose up 
4. Using postman collection. Start firing the REST API calls to test.
